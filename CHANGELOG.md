# 1.2.2

- 修复容器缺少 `zhdate` 时农历静默显示“农历未知”的问题，并在计算失败时记录日志。
- 新番封面渲染前预下载并转换为 base64 data URI，避免 Playwright 截图时因外链超时/防盗链/CDN 波动导致封面缺失。
- 增加 `.gitignore`，排除运行数据、缓存、数据库、日志和本地环境文件，避免敏感配置误提交。

# Changelog

## 1.2.0 - 2026-03-10

### Added
- 新增 `zhdate` 依赖，修复未安装农历库时只能显示“农历未知”的问题。
- 新增 `noto-sans-sc-black.otf` 字体资源，用于标题字体优化。

### Changed
- 重做 `daily_news.html` 的整体版式，使其更贴近最新设计稿。
- 优化“真寻日报”与 `MAHIRO NEWS` 的标题视觉效果与阴影位置。
- 重新调整日期卡、页签、卡片边框、列表字号与行距，提升整体清晰度与一致性。
- 优化正文与标签字体策略，提升生成图片的规范性与可读性。

### Fixed
- 修复模板预览时可能出现的未渲染样式与内容展示问题。
- 修复因依赖缺失导致的农历日期回退问题。
